# Stamp

Stamp is a Python package that provides structured data handling, internal routing utilities, and supporting modules used across the project. Version **4.11** is the first release published with full source code on GitLab.

Unfortunately Stamp is not yet on PyPI

## Project Structure

```
lib/                Core modules
tests/              Test files
info/changelogs/    Version and change history
docs/               Documentation stubs
dist/               Build artifacts (wheel, tar.gz)
stamp.egg-info/     Package metadata
pyproject.toml      Build configuration
```

## Installation

Install from the wheel in the `dist/` directory:

```
pip install stamp-4.11-py3-none-any.whl
```

You can also do this if our are still in the dist/ directory

```
pip install .
```
